## 安装说明

### Android 手机/平板

1. 下载 APK 文件到设备
2. 点击 APK 文件
3. 允许"未知来源"安装
4. 等待安装完成

### Android TV / 电视盒子

1. 下载 APK 文件
2. 通过 USB 驱动器复制到设备
3. 在电视上使用文件管理器
4. 导航到 APK 文件并安装

5. 或者使用 ADB:
   ```bash
   # 连接电视到同一网络
   adb connect <电视IP>:5555
   
   # 安装 APK
   adb install -r lunatv-android.apk
   ```
